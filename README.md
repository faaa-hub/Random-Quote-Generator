# 💬 Random Quote Generator

Генератор случайных цитат — GUI-приложение для просмотра, добавления и фильтрации цитат.

---

## 📋 Описание

Приложение позволяет:
- ✅ Генерировать случайные цитаты из коллекции
- ✅ Добавлять новые цитаты с указанием автора и темы
- ✅ Просматривать историю сгенерированных цитат
- ✅ Фильтровать историю по автору и теме
- ✅ Сохранять коллекцию и историю в JSON-файлы

## 📦 Требования

### Версия Python
- **Python 3.6** или выше
- Рекомендуется Python 3.8+

### Зависимости
Все используемые модули входят в **стандартную библиотеку Python**:

| Модуль | Описание |
|--------|----------|
| `tkinter` | Графический интерфейс (встроен) |
| `json` | Работа с JSON файлами (встроен) |
| `os` | Работа с файловой системой (встроен) |
| `random` | Генерация случайных чисел (встроен) |
| `unittest` | Тестирование (встроен) |

> ⚠️ **Примечание:** На некоторых Linux-системах tkinter может требовать отдельной установки:
> ```bash

> sudo apt-get install python3-tk
